# AlignTrue Documentation

**Make your AI coding assistants follow your project's standards.** AlignTrue converts simple YAML rules into agent-specific formats for Cursor, GitHub Copilot, Claude Code, and 25+ other AI agents.

## Why AlignTrue?

- **Single source of truth** - Write rules once, export to all your AI agents
- **Version controlled** - Track rule changes in git like any other code
- **Team collaboration** - Share rules across your team with lockfiles and drift detection
- **28+ AI agents** - Cursor, Copilot, Claude Code, Aider, Windsurf, and more
- **Local-first** - Everything works offline, cloud is optional
- **Deterministic** - Same rules produce identical outputs every time

## Quick Start

Get started in under 60 seconds:

```bash
npm install -g @aligntrue/cli
aligntrue init
aligntrue sync
```

Read the [Quickstart Guide](/getting-started/quickstart) for detailed instructions.
